# 运行示例
## 1. 安装全部依赖
凡是有 package.json 的地方都需要安装依赖。
npm i 或 yarn

## 2. 打包 topology.js
执行最外层 package.json 中的 bulid 命令。
cd 到最外层， npm run build or yarn build
确保执行后 dist/topology/topology.js 存在


## 3. 运行示例项目
根据个人擅长或喜好，运行示例项目。
### es5
推荐使用 live server.

### vue
安装依赖，npm run dev 或 yarn dev.

### react 
安装依赖，命令可查看 package.json.
